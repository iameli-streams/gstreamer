amfav1enc

amfav1enc element encodes raw video stream into compressed AV1 bitstream via AMD AMF API.

Example launch line

gst-launch-1.0 videotestsrc num-buffers=100 ! amfav1enc ! av1parse ! webmmux ! filesink location=encoded.webm
